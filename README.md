# beauty-book--front

Next.js 16 + React 19 + Tailwind v4 기반 프론트엔드.

## Dev

```bash
npm install
npm run dev   # http://localhost:3100
```

## Env

- `NEXT_PUBLIC_API_URL` — 백엔드 API URL (기본 `http://localhost:4101`)

## Ports

| 서비스 | 포트 |
| --- | --- |
| Front (Next.js) | 3100 |
| Backend (Spring Boot) | 4101 |
| Postgres (host) | 5434 |
